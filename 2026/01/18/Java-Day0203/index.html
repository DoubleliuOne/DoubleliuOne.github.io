Java学习笔记 Day02：变量与数据类型详解

发表于2026-01-18|更新于2026-01-18|Java学习路
|浏览量:

变量

变量是程序的基本单位，表示内存中的一个区域。使用变量必须遵循 先声明再使用 的原则。

三个基本要素：类型 + 名称 + 值

public class main{
  public static void main(String[] args){
     int age = 30;
     double score = 88.9;
     char gender = '男';
     String name = 'king';
     System.out.println(name);
  } 
}
数据类型
基本数据类型
数值型：进行相等判断时，若是小数，应判断差值的绝对值是否在精度范围内。

字符型 (char)：

用单引号 ''。

本质是一个整数 (Unicode码)，可以进行运算。

布尔型 (boolean)：占一个字节，只有 true 和 false。

引用数据类型
包含：类、接口、数组。

类型转换
自动类型转换
容量小的数据类型自动转换为容量大的数据类型。

byte, short, char 之间不会自动转换，它们计算时首先转换为 int。

boolean 不参与转换。

强制类型转换
将容量大的转换为容量小的，使用 (类型) 强制转换符，可能造成精度降低或溢出。

String与基本类型转换
基本转String：基本类型的值 + ""

String转基本：调用包装类的 parse 方法 (如 Integer.parseInt)。

# 运算符

## 1. 算术运算符
包括 `+`, `-`, `*`, `/`, `%` (取模), `++`, `--`。
> 注意：Java中 `%` 的本质公式为 `a % b = a - a / b * b`。

## 2. 赋值运算符
- 基本赋值：`=`
- 复合赋值：`+=`, `-=`, `*=` 等。
- **细节**：复合赋值运算符会进行隐式的类型转换 (例如 `byte b = 2; b += 3;`)。

## 3. 关系运算符
结果都是 `boolean` 型 (`true` 或 `false`)。包括 `==`, `!=`, `<`, `>`, `<=`, `>=`, `instanceof`。

## 4. 逻辑运算符
- `&` (逻辑与) vs `&&` (短路与)：`&&` 效率更高，如果第一个条件为 false，后面不再判断。
- `|` (逻辑或) vs `||` (短路或)：`||` 效率更高，如果第一个条件为 true，后面不再判断。
- `!` (取反)
- `^` (异或)：不同为 true，相同为 false。

## 5. 三元运算符
语法：`条件表达式 ? 表达式1 : 表达式2;`

# 标识符命名规范

1. **包名**：所有字母小写 (如 `com.hsp.crm`)。
2. **类名/接口名**：大驼峰 (如 `TankShotGame`)。
3. **变量名/方法名**：小驼峰 (如 `tankShotGame`)。
4. **常量名**：全大写，下划线连接 (如 `TAX_RATE`)。
